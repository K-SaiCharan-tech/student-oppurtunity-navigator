# Student Opportunity Navigator 🎯

A multi-agent AI system that helps students discover the right opportunities, create personalized action plans, and match their profile to the best career paths.

## Built With
- n8n (workflow automation)
- Featherless AI (DeepSeek-V3 model)
- 3 Collaborative AI Agents

## The Problem
Students waste hours searching for internships, hackathons and certifications with no guidance. There is no intelligent system that understands their profile and creates a personalized roadmap.

## The Solution
Three AI agents working together:

### Agent 1 — Career Advisor Agent
Finds the best internships, hackathons and certifications based on the student's skills and interests.

### Agent 2 — Action Planner Agent  
Creates a detailed week-by-week action plan for the student to apply and prepare for success.

### Agent 3 — Profile Matcher Agent
Matches the student's skills and background to the most suitable opportunities and explains why each one fits perfectly.

## How It Works
1. Student shares their skills and interests in the chat
2. Career Advisor Agent finds relevant opportunities
3. Action Planner Agent creates a personalized weekly plan
4. Profile Matcher Agent matches the student to best-fit roles
